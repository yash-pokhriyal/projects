Redux terminology 

1. redux store : jahan pe sara ka saara samaan rkhenge saara state manage krenge -->saari functionality ko yhan rkhna 
2. reducers: ye aapke function hote hain . Increse by 1 and decrease by 1 and so on ....
3. slice : maanein tukda : reducer inke andar aate hain | saari functionality state manage yhan bnaya jaega .
4. useSelector : iski madad se select krte hain 
5. dispatch : iski help se bhejenge 
selector se select krenge and dispatch se bol skte use krenge
